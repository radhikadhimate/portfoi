RESPONSIVE WEB PAGE


This project is a responsive web page designed to showcase information about Chania,
a city on the Greek island of Crete. The webpage layout dynamically adapts based on screen size,
ensuring a seamless user experience across devices.

Key Features
	1.	Responsive Layout:
	•	The page adjusts when resized, demonstrating the use of CSS media queries.
	•	Different sections reorganize to provide optimal readability on desktops, tablets, and mobiles.
	2.	Navigation Sidebar:
	•	A left-aligned navigation menu with buttons labeled:
	•	The Flight
	•	The City
	•	The Island
	•	The Food
	•	This sidebar is styled for easy access and visibility.
	3.	Content Section:
	•	The main content area provides details about Chania, including its historical and modern significance.
	•	Uses structured typography for readability.
	4.	Information Box (Right Sidebar):
	•	A visually distinct box provides quick facts such as:
	•	What? – Explains Chania as a city in Crete.
	•	Where? – Mentions Crete’s location in the Mediterranean.
	•	How? – Details about travel accessibility.
	5.	Footer Section:
	•	A blue footer at the bottom guides users to resize the browser window and observe how the layout adapts, highlighting the page’s responsiveness.

Technologies Used
	•	HTML: Structure of the webpage.
	•	CSS: Styling and responsive design.
	•	Media Queries: Ensuring proper layout adjustments.

This project is a great example of how to design an engaging and user-friendly responsive webpage,
making it accessible on different screen sizes without compromising usability.
